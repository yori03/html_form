Jika Anda mengklik tombol "Kirim", data formulir akan dikirim ke halaman bernama "/action_page.php".

sebuah formulir sederhana yang memungkinkan pengguna untuk memasukkan nama depan dan belakang. Formulir ini memiliki dua input teks untuk nama depan dan belakang, yang sudah diisi dengan nilai default "Budi" dan "Prawira" secara berturut-turut. Ketika tombol "Kirim" diklik, data dari formulir akan dikirim ke halaman bernama "/action_page.php". Selain itu, terdapat tombol "Reset" yang digunakan untuk menghapus data yang telah dimasukkan dan mengembalikan nilai input ke nilai default.
